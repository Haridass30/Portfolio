# Skill Logos

Please add your skill logo images to this folder.

## Instructions:
1. Add PNG or SVG files for each technology
2. Name them according to the technology (e.g., `aws.png`, `docker.png`, `kubernetes.png`)
3. Recommended size: 200x200px or similar aspect ratio
4. Transparent background works best

## Required Logos:
- aws.png
- docker.png
- kubernetes.png
- nodejs.png
- python.png
- java.png
- mysql.png
- github.png
- linux.png
- redis.png
- nginx.png
- grafana.png

You can download logos from:
- https://simpleicons.org/
- https://www.vectorlogo.zone/
- Official brand websites
